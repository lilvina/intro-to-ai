# Intro to Building with AI APIs

This project is a beginner-friendly curriculum designed to teach developers how to build applications using AI APIs.

## What You Will Learn
- How APIs work
- How to make API requests
- How to use AI APIs
- Prompt engineering basics
- Building real AI-powered features

## Lessons
1. API Basics
2. First AI API Call
3. Prompt Design
4. Build a Feature
5. Error Handling

## Final Project
Build a working AI-powered CLI application.

## Tech Stack
- Python
- OpenAI API
- Requests

## How to Run

```bash
pip3 install -r requirements.txt
python3 lesson-2-ai-api-call/ai_request.py
